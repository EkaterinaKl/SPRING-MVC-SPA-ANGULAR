О программе

SPA: Angular JS + Spring

Angular JS 1.5:

  • использован ngResource
  • использован uiRouter
  • использован Angular Bootstrap
  • использован Angular Bootstrap UI
  • разработан notifyService для MsgBox
  • разработана директива в notifyService
  • компонент для CRUD-операций
  • фильтр товаров по наимненованию
  • локализация валюты
  • сортировка

Spring

  • класс для обработки ошибок JSON
  • константы ошибок вынесены в отдельный класс
  • обработка маршрутизации Angular - stateProvider при обновлении страницы
    (stateProvider может быть использован в дальнейшем для авторизации в отличии от обычного провайдера)
